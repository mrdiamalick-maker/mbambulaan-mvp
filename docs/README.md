## Positionnement

Mbàmbulaan est un écosystème numérique au service de la pêche artisanale sénégalaise. La plateforme vise à connecter les arrivages, les besoins, les opportunités, les réservations, les transactions, les notifications et les indicateurs territoriaux dans un parcours cohérent.

## Méthode de développement

Le projet avance par packs de développement. Un pack doit livrer une capacité métier complète, pas une micro-tâche technique.
