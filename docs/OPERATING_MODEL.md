# Operating model Mbambulaan

## Principe

Mbambulaan est pilote comme un produit, pas comme une simple maquette.

Chaque decision doit renforcer la demonstration du parcours metier.

## Roles

### Fondateur

- porte la vision
- valide la valeur metier
- arbitre les priorites
- prepare les discussions terrain

### Product lead

- structure la roadmap
- transforme les idees en packs
- protege le cap produit
- verifie la coherence du MVP

### Codex

- developpe par packs complets
- modifie le code
- lance les validations techniques
- met a jour la PR

### GitHub

- garde la trace du produit
- centralise documentation et code
- sert de source de verite

## Regle de developpement

Un pack egale une capacite metier complete.

Pas de micro sprint.

Pas de fonctionnalite isolee sans valeur de demonstration.
