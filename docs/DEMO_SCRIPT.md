# Script de démonstration — Mbàmbulaan

## Objectif

Présenter Mbàmbulaan en 3 à 5 minutes comme un écosystème numérique au service de la pêche artisanale sénégalaise.

La démonstration doit raconter une histoire, pas seulement parcourir des pages.

## Message d’ouverture

Mbàmbulaan rend visibles les arrivages, connecte les besoins, génère des opportunités, suit les transactions et donne une lecture territoriale de la filière.

## Déroulé recommandé

### 1. Vision

Ouvrir la landing page.

Message clé :

> Mbàmbulaan n’est pas une simple marketplace. C’est un système de coordination pour la filière halieutique artisanale.

### 2. Démo guidée

Ouvrir `/demo`.

Lancer la démonstration.

Montrer le parcours complet :

- un pêcheur débarque un lot ;
- l’arrivage devient visible ;
- un besoin compatible existe ;
- le moteur détecte une opportunité ;
- un acteur réserve ;
- une transaction est générée ;
- les notifications sont envoyées ;
- le dashboard et la carte reflètent l’impact.

### 3. Arrivages

Ouvrir `/arrivages`.

Montrer :

- liste des lots ;
- recherche ;
- filtre ;
- déclaration d’un arrivage.

### 4. Besoins

Ouvrir `/besoins`.

Montrer :

- expression de la demande ;
- niveau d’urgence ;
- priorisation.

### 5. Opportunités

Ouvrir `/opportunites`.

Montrer :

- score de recommandation ;
- confiance acteur ;
- priorité ;
- détail d’une opportunité.

### 6. Transaction

Ouvrir `/transactions`.

Montrer que Mbàmbulaan ne s’arrête pas à la mise en relation : la plateforme suit le déroulement de l’échange.

### 7. Dashboard

Ouvrir `/dashboard`.

Montrer :

- impact de la journée ;
- tensions territoriales ;
- priorités ;
- alertes ;
- indicateurs d’impact.

### 8. Carte des quais

Ouvrir `/quais`.

Montrer la dimension territoriale.

Message clé :

> Mbàmbulaan donne une lecture en temps réel des flux, des tensions et des opportunités sur les quais.

## Conclusion

Le MVP fonctionne avec des données simulées.

La V1 pourra connecter ces modules à des données réelles, des comptes utilisateurs, une API, une base de données et des partenaires terrain.
