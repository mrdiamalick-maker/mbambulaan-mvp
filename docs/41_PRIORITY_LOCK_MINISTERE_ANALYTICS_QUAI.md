La prochaine passe est utile seulement si elle ameliore :

- filtre dynamique par quai ;
- KPIs qui changent selon le quai ;
- visualisations type donut, barres, jauges ou progress bars ;
- referents terrain par quai ;
- actions visibles dans l'interface ;
- bloc Actions en attente ;
- IA contextualisee au quai selectionne ;
- palette mer et rendu analytics premium.
