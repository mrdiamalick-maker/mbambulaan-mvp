# Template Shortlist Mbàmbulaan

## Objectif

Choisir un template premium pour reconstruire Mbàmbulaan comme infrastructure de coordination, command center métier et espace de travail par rôle.

La PR 13 reste un laboratoire. Elle ne doit pas devenir la base finale.

## Critères de scoring

| Critère | Points |
| --- | ---: |
| Qualité visuelle premium | 15 |
| Dashboard / command center | 15 |
| Tables et data display | 10 |
| Charts et visualisations | 10 |
| Sidebar / navigation | 10 |
| Forms / flows commerciaux | 8 |
| Pages détails / profils | 8 |
| Adaptation role-based | 8 |
| Responsive | 6 |
| Facilité d'intégration Next / React | 6 |
| Licence / coût / maintenabilité | 4 |

Score minimum recommandé : 80 / 100.

## Shortlist

| Rang | Template | Score | Décision |
| ---: | --- | ---: | --- |
| 1 | TailAdmin | 88 | Meilleur compromis vitesse, stack et composants |
| 2 | Metronic | 86 | Plus puissant, mais plus lourd |
| 3 | Mosaic | 84 | Très propre, sobre, bon choix premium |
| 4 | Materio | 79 | Solide mais moins aligné Tailwind |
| 5 | Tremor + shadcn custom | 76 | Flexible mais demande plus de design |

## 1. TailAdmin

Type : admin dashboard Tailwind multi-framework.

Pourquoi il convient : Next.js, React, Tailwind, dashboards analytics, CRM, SaaS, logistics, finance, sidebar, charts, maps, tables et cards.

Forces : vitesse d'intégration, stack proche, composants utiles pour espace privé, flux, territoires, coordination.

Faiblesses : risque de rendu générique si la personnalisation est faible ; carte territoire à adapter.

Recommandation : candidat prioritaire.

## 2. Mosaic

Type : Tailwind admin dashboard premium.

Forces : sobre, mature, lisible, crédible.

Faiblesses : moins command center métier ; carte opérationnelle à adapter.

Recommandation : alternative premium sobre.

## 3. Metronic

Type : toolkit dashboard très complet.

Forces : richesse exceptionnelle, nombreux layouts, profils, CRM, projets, settings.

Faiblesses : plus lourd, risque de complexité et de dispersion.

Recommandation : alternative très complète si la richesse produit prime sur la vitesse.

## 4. Materio

Type : Next.js admin template basé sur Material UI.

Forces : structuré, robuste, adapté aux flows SaaS.

Faiblesses : moins naturel pour un repo Tailwind et une identité visuelle propriétaire.

Recommandation : candidat secondaire.

## 5. Tremor + shadcn custom

Type : component libraries.

Forces : flexibilité et identité sur mesure.

Faiblesses : pas de pages complètes prêtes ; effort design plus fort.

Recommandation : option à garder si aucun template complet ne convient.

## Décision proposée

Priorité : TailAdmin.

Alternative sobre : Mosaic.

Alternative riche : Metronic.

Avant achat, faire une revue visuelle rapide de TailAdmin, Mosaic et Metronic.
