# User stories Mbambulaan

## Pecheur

En tant que pecheur, je veux declarer un arrivage pour rendre mon lot visible.

En tant que pecheur, je veux suivre les reservations pour savoir si mon lot interesse un acheteur.

En tant que pecheur, je veux recevoir une notification quand une opportunite existe.

## Mareyeur

En tant que mareyeur, je veux consulter les arrivages pour trouver rapidement des lots disponibles.

En tant que mareyeur, je veux publier un besoin pour recevoir des recommandations.

En tant que mareyeur, je veux reserver un lot pour securiser mon achat.

## Transformateur

En tant que transformateur, je veux publier un besoin pour planifier mon approvisionnement.

En tant que transformateur, je veux voir les opportunites prioritaires pour reduire les ruptures.

## Collectivite

En tant que collectivite, je veux voir les tensions par quai pour prioriser les actions.

En tant que collectivite, je veux suivre les indicateurs d impact pour comprendre la valeur creee.

## Administration

En tant qu administration, je veux disposer d une vue synthetique pour orienter les decisions publiques.
