- La landing explique clairement la promesse.
- La demo raconte un parcours complet.
- Les arrivages sont visibles.
- Les besoins sont visibles.
- Les opportunites sont generees.
- Une reservation peut etre comprise.
- Une transaction peut etre suivie.
- Les notifications donnent une action utile.
- Le dashboard montre l impact.
- La carte donne une lecture territoriale.

## Intelligence metier

- Le matching cree une opportunite claire.
- La recommandation explique pourquoi une opportunite est pertinente.
- Le score de confiance est visible.
- L impact est visible.
- La tension territoriale est visible.
- La priorisation est visible.
- Les alertes sont actionnables.
- La tracabilite est finalisee apres reprise Codex.

## Demonstration

- La demo tient en moins de cinq minutes.
- Le message marketplace est evite.
- Le message ecosysteme de coordination est clair.
- Les donnees paraissent coherentes.
- Les modules sont lies entre eux.

## Decision

Le MVP est pret a presenter quand la tracabilite est finalisee et que le build est valide.
