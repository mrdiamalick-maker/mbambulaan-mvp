# Direction images et visuels premium - Mbambulaan

## Decision CPO

Oui, il faut exploiter des images et visuels, mais pas n'importe comment.

Mbambulaan ne doit pas utiliser des photos decoratives simplement parce qu'elles sont jolies. Les images doivent renforcer la promesse : territoire, coordination, donnees, acteurs terrain, decision institutionnelle.

## Probleme a eviter

- images stock generiques ;
- photos de peche trop touristiques ;
- images qui donnent une impression ONG humanitaire ;
- photos difficiles a expliquer ;
- visuels trop decoratifs sans lien produit ;
- surcharge visuelle qui affaiblit la lecture analytique.

## Direction recommandee

Combiner trois familles visuelles :

1. Images terrain sobres
2. Visualisations analytiques premium
3. Illustrations cartographiques abstraites

## 1. Images terrain sobres

Usage : landing, cas d'usage, contexte peche artisanale.

Types d'images :

- quai de peche au Senegal ;
- pirogues vues de haut ou de cote ;
- marche ou zone de mareyage ;
- caisse de poisson, froid, transport ;
- silhouettes d'acteurs terrain ;
- mains, carnets, coordination, reunion terrain.

Style attendu :

- photo realiste ;
- lumiere naturelle ;
- couleurs mer, sable, bois, turquoise ;
- pas trop chargee ;
- pas misere ;
- pas carte postale ;
- pas folklorisation.

Message : Mbambulaan part du reel.

## 2. Visualisations analytiques premium

Usage : hero, apercu solution, espace Ministere.

Types de visuels :

- carte stylisee du littoral ;
- points de quais ;
- flux entre acteurs ;
- donuts, barres, scores, mini charts ;
- panneaux analytics compacts ;
- overlays de donnees sur carte ;
- parcours Signal -> Analyse -> Coordination -> Decision.

Style attendu :

- propre ;
- institutionnel ;
- Qlik-like sans copier ;
- ocean, turquoise, sable, lagon, ambre, corail ;
- beaucoup d'espace ;
- peu de texte ;
- data lisible.

Message : Mbambulaan transforme le reel en decision.

## 3. Illustrations cartographiques abstraites

Usage : carte, sections cartographie, espace Ministere.

Types de visuels :

- littoral abstrait ;
- formes de courant marin ;
- lignes de coordination ;
- points de tension ;
- halos par niveau de priorite ;
- grilles legeres de donnees ;
- cartes sans etre trop geographiques.

Style attendu :

- premium ;
- explicable ;
- sobre ;
- pas gadget ;
- pas decoration pure.

Message : Mbambulaan organise les territoires.

## Images a privilegier dans la landing

### Hero

Ne pas mettre une grande photo brute.

Recommandation : visuel hybride.

Fond doux inspire mer + carte stylisee + mini modules analytics + point quai selectionne.

Texte autour : Ecosysteme numerique au service de la peche artisanale.

### Section flow

Utiliser un visuel clair :

Signal terrain -> Analyse territoriale -> Coordination -> Decision / Rapport.

Chaque etape peut avoir une icone ou mini image.

### Cas d'usage

Utiliser une matrice visuelle, pas des photos partout.

Acteur -> besoin -> module -> decision facilitee.

### Ministere

Utiliser une image ou illustration institutionnelle : carte + indicateurs + note en brouillon.

## Images a privilegier dans l'espace Ministere

Dans l'application, ne pas mettre de grandes photos.

L'espace Ministere doit rester analytique.

Utiliser plutot :

- mini carte ;
- icones quai ;
- avatars sobres des referents ;
- badges ;
- mini charts ;
- fiches compactes.

Pour les referents terrain, utiliser avatars generiques sobres ou initiales, pas des photos reelles sans consentement.

## Recommandation assets MVP

A court terme, eviter les images externes non maitrisees.

Utiliser :

- SVG internes ;
- gradients mer ;
- cartes stylisees CSS/SVG ;
- avatars generiques ;
- icones simples ;
- placeholders premium.

A moyen terme, prevoir une vraie direction photo terrain avec droits et consentement.

## Prompt image IA possible

Prompt pour visuel hero :

Une interface analytique premium pour une plateforme institutionnelle de coordination de la peche artisanale senegalaise, carte stylisee du littoral, points de quais, visualisations data sobres, palette bleu ocean turquoise sable vert lagon ambre, style B2B moderne, beaucoup d'espace, aucun texte lisible, pas de logo, pas de personnes reconnaissables.

Prompt pour visuel carte :

Carte abstraite premium du littoral senegalais avec points de quais de peche, halos de priorite, lignes de coordination, overlays de donnees, style analytics institutionnel, palette mer turquoise sable corail, sobre, moderne, lisible, sans texte.

Prompt pour referents :

Avatars generiques sobres pour referents terrain peche artisanale, style institutionnel moderne, silhouettes non reconnaissables, palette mer, fond simple, pas de realisme identitaire.

## Regles de securite et confiance

- pas de photo d'une personne reconnaissable sans consentement ;
- pas d'image qui suggere une donnee reelle non verifiee ;
- pas de photos externes sans controle des droits ;
- pas d'image qui infantilise ou misere les acteurs ;
- pas de decoration sans fonction.

## Prochaine instruction Codex

Ajouter des placeholders visuels premium sans importer d'images externes.

Priorite :

- hero landing ;
- flow Signal -> Analyse -> Coordination -> Decision ;
- apercu carte ;
- avatars generiques referents ;
- visualisations KPI type donut/bar/progress.

Les visuels doivent renforcer le produit, pas l'alourdir.
