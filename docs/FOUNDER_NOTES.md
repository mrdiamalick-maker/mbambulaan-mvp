# Notes fondateur

## Cap

Mbambulaan doit rester un produit de coordination pour la peche artisanale senegalaise.

Le projet ne doit pas etre reduit a une marketplace.

## Promesse

Rendre visibles les arrivages, connecter les besoins, detecter les opportunites, suivre les transactions et mesurer l impact.

## Discipline produit

Une nouvelle idee doit etre acceptee seulement si elle renforce une capacite metier utile.

## Priorite actuelle

## Message a garder

Mbambulaan aide les acteurs a mieux voir, mieux coordonner et mieux decider.
