# Dataset mock v1 - Quais Ministere

## Objectif

Donner a Codex une matiere concrete pour construire une experience dynamique par quai.

Le but n'est pas d'avoir des donnees parfaites. Le but est de simuler une logique produit credible : choisir un quai, comprendre la situation, voir les referents, declencher une action et garder une trace.

## Regle produit

Chaque quai doit raconter une histoire differente.

Si tous les quais se ressemblent, le filtre dynamique n'a pas de valeur.

Chaque quai doit donc avoir :

- un niveau de tension ;
- un risque dominant ;
- des KPIs specifiques ;
- des programmes differents ;
- des budgets differents ;
- des incidents differents ;
- des ressources critiques differentes ;
- des referents terrain ;
- des preuves ;
- une action recommandee.

## Structure de donnees recommandee

```ts
type QuayProfile = {
  id: string;
  name: string;
  commune: string;
  region: string;
  lat: number;
  lng: number;
  tension: "Critique" | "Forte" | "Moyenne" | "Faible";
  priorityScore: number;
  mainRisk: string;
  lastUpdate: string;
  metrics: {
    openIncidents: number;
    budgetExecution: number;
    pendingFundingFcfa: number;
    criticalResources: number;
    activePrograms: number;
    validatedProofs: number;
    availableReferents: number;
  };
  programs: QuayProgram[];
  budget: QuayBudget;
  incidents: QuayIncident[];
  resources: QuayResource[];
  referents: QuayReferent[];
  proofs: QuayProof[];
  recommendedAction: string;
};
```

## Logique UX attendue

Quand un quai est selectionne :

- la carte met le quai en evidence ;
- les KPIs changent ;
- les charts changent ;
- la fiche quai change ;
- les referents changent ;
- les actions recommandees changent ;
- l'IA produit une synthese courte du quai ;
- les actions en attente gardent l'historique des clics.

## Challenge

La solution devient achetable si le Ministere peut passer d'une vision nationale a une action locale en moins de 30 secondes.

C'est ce passage national -> quai -> referent -> action -> trace qui constitue l'avantage Mbambulaan.
